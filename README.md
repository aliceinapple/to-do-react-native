# to-do-react-native

- Clone this repository

- Go to the project folder
```cd to-do-app```

- Install dependencies
```npm install```

- Install mongoDB locally

https://www.mongodb.com/try/download/community

https://metanit.com/nosql/mongodb/1.2.php

- Start the server
```node server.mjs```

- Launch the application
```npx expo start```
